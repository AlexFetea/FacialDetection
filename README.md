# Local Development

## Setting up the frontend

Requirements: `node`, `npm`

Clone the repository and install npm packages in src/main/frontend:
```
cd src/main/frontend
npm install
```
Start the app:
```
npm start
```
Visit `https://localhost:3000/`

Done!
